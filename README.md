## 📋 About thinking-skills

thinking-skills offers 20 human thinking frameworks in the form of Claude Code custom commands. These frameworks include critical thinking, first principles, red team tactics, Bayesian reasoning, and more. Each framework helps you analyze problems, make decisions, and reason more clearly.

The application runs on Windows and requires no prior coding experience. You interact with it through Claude Code, an AI assistant environment. This package includes clear commands that guide your thinking in practical ways.

Topics covered include AI tools, decision making, mental models, prompts for reasoning, and various thinking strategies. The goal is to help you improve reasoning and clarity in your daily tasks.

## 💡 Features

thinking-skills helps you by providing ready-to-use frameworks that improve the way you think and solve problems.

- **Critical Thinking**: Break down problems and evaluate evidence clearly.  
- **First Principles**: Solve issues from the ground up.  
- **Red Team**: Challenge your ideas to find weaknesses.  
- **Bayesian Reasoning**: Update beliefs with new information.  
- **Decision-Making Models**: Structured advice for tough choices.  

Each framework is a separate Claude Code command, so you can use only what you need. You can also combine them for deeper analysis.

## ⚙️ How to Use thinking-skills Commands

1. Open your Claude Code assistant.  
2. Import or load the `thinking-skills` command set.  
3. Enter the name of the framework you want to use, for example, “Critical Thinking.”  
4. Follow the prompts to enter your question or problem.  
5. Receive a structured answer based on that thinking model.

Each command guides you step-by-step through a clear framework. You don’t need to memorize the models yourself.

## 🛠 Troubleshooting

- **Commands don’t run**  
  Ensure Claude Code is properly installed and updated. Verify you added the `thinking-skills` commands to your workspace.

- **Errors during extraction**  
  Use Windows built-in unzip tool or a program like 7-Zip to extract ZIP files.

- **Files missing after download**  
  Double-check the latest release downloaded fully. A stable internet connection helps prevent incomplete downloads.

- **Not sure how to open Claude Code**  
  Refer to your AI assistant’s documentation or help section to launch the command input interface.
